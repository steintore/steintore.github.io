To start using an advanced editors often seems like hell, that's why I've tried to make a short list of the commands I've used most.

  • :sp <filepath> - split the screen and show multiple files. Move between the file with <Ctrl-W> and then the arrow keys
  • :%s/a/b/gc - substitute a with b globally with confirmation for each replacement
